# Country Lookup Tool — Find Country Info by Name, Currency or Timezone

Search 195 countries by name, currency code, or timezone. Get capitals, phone codes, currencies, and languages instantly. Free online tool.

**Read time:** 5 min | **Category:** converters

---

The [Country Lookup](/en/tools/country-lookup) tool on FindUtils lets you search 195 countries by name, currency code, timezone, or phone code and get structured results instantly. Type a query, and the tool returns the capital, currency, phone code, timezone, languages, and region for every matching country -- all in one card you can copy with a single click.

Whether you are filling out international shipping forms, building a localization feature, or simply settling a geography question, this guide walks you through every feature of the country lookup tool and the best ways to use it.

The country lookup tool supports fuzzy matching across multiple data fields, so you do not need to type exact names or codes. It searches country names, ISO codes, capitals, currencies, currency codes, phone codes, timezones, and languages simultaneously.
